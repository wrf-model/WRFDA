WRFDA Version 3.5.1: UPDATES

WRFDA Version 3.5.1 was released on September 23, 2013.

Updated features:

  • This is primarily a bug-fix release. The following bugs have been fixed:
    • All the bugs fixed in the special minor release: Version 3.5.0.1, including updated BUFR libraries to allow WRFDA to read and assimilate NCEP PREPBUFR files.
    • Some BUFRLIB bugs not addressed in the previous release
    • A bug causing failures when "print_detail_outerloop=true"
    • A few more miscellaneous fixes

For instructions on compiling and running the latest version of WRFDA, please refer to the User's Guide
