2.2. Global option in 3D-VAR (Rizvi)

 

The coding work was done. The code is in the old repository. It has been tested on MAC. Some problems were introduced by cpp.

 

The global test-case converges in ~50 iterations. The regional regression tests are not bit-reproducable with the previous version. After investigation, this is found due to a reordering of map-factor calculations in e.g. psichi_to_uv (performed to improve efficiency).

 

The remaining issues:

1)     Transform efficiency of psichi_to_uv.

2)     Vertical velocity computation. A bug fixed.

3)     Vertical transform efficiency (improved if store eigenvector/value product).
