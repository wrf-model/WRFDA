Test case data

Click here to download the most recent test data set. This test data set is designed to work with Version 3.7, but should work with versions as old as 3.5.

This is a zipped .tar file which includes data for the 2008-02-05 test cases mentioned in the User's Guide:

  • A sample be.dat for cv_options=5
  • LITTLE_R format observational data (12Z and 18z)
  • WRFDA-3DVAR and 4DVAR observational data (prepared by OBSPROC)
  • first guess and boundary input (12Z and 18z)
  • AMSU-A and AMSU-B satellite radiance observational data (12Z and 18z)
  • 6-hour accumulated rainfall data (valid 18z)

This test case data is for training and testing purposes only.

For cv_options=5 training purposes, you can download the pre-generated 1-day forecasts and use the gen_be utility to generate the background error statistics file.

To learn more about WRFDA and how to use it, please see the User's Guide.
